# HTML Basics – Tables, Forms and Frames

## Project Overview

This repository contains simple HTML files created as part of Full Stack Development (FSD) practice.

The project demonstrates the basic usage of:

- Tables
- Forms
- Frames / Iframes

All files are developed using pure HTML to understand fundamental webpage structure and user interaction concepts.


## Objectives

- Learn how to create tables in HTML
- Understand how to design forms for user input
- Practice embedding content using frames or iframes
- Strengthen basic HTML knowledge


## Technologies Used

- HTML5


## Project Structure

```
HTML-Tables-Forms-Frames/
├── table.html
├── form.html
├── frame.html
└── README.md
```


## File Description

### 1. table.html
Demonstrates how to create tables using:
- table tag
- tr (table row)
- th (table header)
- td (table data)
- rowspan and colspan attributes

### 2. form.html
Demonstrates how to create forms using:
- form tag
- input fields (text, password, email, radio, checkbox)
- textarea
- select and option
- submit button

### 3. frame.html
Demonstrates how to use iframe to embed:
- Another webpage
- External websites
- Internal HTML pages


## How to Run

1. Clone the repository:

```
git clone <repository-link>
cd HTML-Tables-Forms-Frames
```

2. Open any HTML file in a web browser.

No additional software or dependencies are required.

- Knowledge of form elements and user input handling
